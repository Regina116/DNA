# DNA

Распознавание личности по ДНК

Имеется база ДНК различных людей в файлах small.csv и large.csv
Также имеем текстовые файлы с ДНК людей, которых нужно распознать (1.txt - 20.txt)
С помощью программы находится соответствие ДНК и выдается имя человека.


Для запуска программы необходимо:
1. Открыть терминал
2. Перейти в текущую директорию
3. Ввести: python dna.py "нужная_база" "кого_ищем"

НАПРИМЕР ВВОД: python dna.py large.csv 6.txt
